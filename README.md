# Desafío 2 – Condiciones

Este proyecto corresponde al **Desafío 2 del módulo de JavaScript** de la carrera **Full Stack Developer** en **Desafío Latam**.

El objetivo del desafío es aplicar **manipulación básica del DOM** y el uso de **condiciones (`if`, `else if`, `else`)**, resolviendo tres ejercicios independientes.

## Ejercicios desarrollados

### 1. Imagen con borde
Al hacer click sobre una imagen se agrega un borde rojo de 2 pixeles.  
Al hacer click nuevamente, el borde se elimina.

### 2. Selección de stickers
Se permite seleccionar hasta **10 stickers** entre **3 tipos distintos**.  
Si la suma es menor o igual a 10, se muestra la cantidad de stickers seleccionados.  
Si la suma es mayor a 10, se muestra el mensaje **"Llevas demasiados stickers"**.

### 3. Verificación de password
Se utilizan **3 select** para ingresar un password de tres dígitos.  
- Password **911** → “password 1 correcto”  
- Password **714** → “password 2 correcto”  
- Cualquier otro valor → “password incorrecto”

## Tecnologías utilizadas
- HTML5  
- JavaScript (nivel inicial)

> En este desafío el foco está en la lógica y el uso de condiciones, no en el diseño visual.
